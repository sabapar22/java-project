## Problem Statement
Most people know climate change is real, but tracking individual carbon footprints is tedious and depressing. Existing calculators are just forms that give you a number at the end. There is no motivation to keep coming back or to improve daily habits.

## Scope of the Project
This project is a desktop-based application that runs on the command line. It focuses on:
* Tracking daily user activities (Transport, Digital usage, Health).
* Converting these activities into a unified "XP" score.
* Providing immediate feedback to the user about their eco-status.
* Storing data locally so users can track progress over time.

## Target Users
* **Students:** Who want a fun way to track their habits.
* **Eco-conscious individuals:** Who want to see the impact of their daily choices (like driving vs. walking).

## High-Level Features
1.  **Gamification Engine:** Converts real-world metrics (km, hours) into game points.
2.  **Modular Activity System:** The code is written so new activities (like "Eating Meat") can be added easily later.
3.  **Persistence:** Automatic file management to save user history.
